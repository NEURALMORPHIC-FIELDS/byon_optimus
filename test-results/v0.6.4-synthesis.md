### v0.6.4b — Center-coherent coagulation harness

**Code added:**
- `scripts/byon-coagulation-harness.mjs` — drives 60 events on a single center (`byon::execution_boundary`) with varied-but-coherent paraphrases (20-statement pool).
- Per-cycle metric capture: S_t, AR_t, κ_t, Z_norm, ΔX_norm, ω, newly_coagulated.
- Bottleneck auto-diagnosis from factor decomposition.

**Result (clean baseline, no dedup):**

| Metric | min | mean | max |
|---|---|---|---|
| S_t | 0.0147 | 0.0418 | **0.1119** |
| AR | 0.715 | 0.853 | **1.000** |
| κ | 0.172 | 0.431 | 0.979 |
| Z_norm | 1.287 | 7.221 | **31.305** |
| ΔX_norm | 2.002 | 2.002 | 2.002 |

**Coagulation:** 0 events. S_t never crosses θ_s=0.28. Longest streak = 0.

**Bottleneck diagnosed:** `B_t (residue stability) — Z accumulates`.

```
S_t = AR · κ · I_t · B_t
    = 0.853 · 0.431 · I_t · B_t (mean)
    ≈ 0.042

B_t proxy = 1 / (1 + 7.22) = 0.122
→ I_t proxy ≈ 0.042 / (0.853 · 0.431 · 0.122) = 0.94 (healthy)
```

So **B_t = 0.12** is the dominant suppressor of S_t. The user's prediction is exactly confirmed: "câmpul conversațional este prea variabil/rezidual ca să permită coagulare Ω în condiții naturale".

### v0.6.4c — Assimilation/residue tuning

**Code added in `fcem_backend.py`:**
- Per-center embedding LRU history (default size 20).
- Coherent-repeat detection: cosine similarity > 0.92 between new event and history.
- Two strategies tested.

**Attempt 1 — suppress symbolic write entirely on coherent repeat:**

| Metric | v0.6.4b baseline | v0.6.4c attempt 1 |
|---|---|---|
| AR mean | 0.853 | **0.441** ⚠ regressed |
| κ mean | 0.431 | 0.411 |
| Z_norm max | 31.3 | **8.57** ✓ |
| S_t max | 0.112 | 0.153 |
| Bottleneck | B_t | **AR** ⚠ now the limiter |

**Key insight from attempt 1:** AR is built by **symbolic slot_event** writes, NOT by the v0.6.3 numerical companion. The numerical write goes to tf_engine bank but does not appear to register as a perspective event in FCE-Ω observer. Suppressing symbolic kills AR.

**Attempt 2 — keep symbolic but use STABLE label (anchor slot) for coherent repeats:**

| Metric | v0.6.4b | v0.6.4c attempt 2 |
|---|---|---|
| AR mean | 0.853 | 0.851 |
| κ mean | 0.431 | 0.433 |
| Z_norm max | 31.3 | 31.305 (unchanged) |
| S_t max | 0.112 | **0.153** (+37%) |
| Bottleneck | B_t | B_t |

S_t peak improved by 37%, but Z still accumulates the same way. The slot_event itself adds residue regardless of label.

**Attempt 3 — 20 unique events (less repetition, more diversity):**

| Metric | 60 events | 20 events |
|---|---|---|
| AR mean | 0.853 | 0.759 |
| κ mean | 0.431 | **0.311** (lower) |
| Z_norm max | 31.3 | 5.76 |
| S_t max | 0.112 | 0.153 |
| Bottleneck | B_t | **κ** (low coherence) |

Catch-22: **with repetition, Z explodes; with unique events, κ drops.** No combination got S_t over θ_s=0.28 for any cycle, let alone 12 consecutive.

### Why Level 3 is structurally unreachable in current pipeline

The user's two predictions, both confirmed:

1. **"Limitatorul real este produsul I_t · B_t."** — Confirmed: B_t collapses to ~0.12 due to Z_norm reaching ~31 on 60 repeated coherent events. I_t stays ~0.94.

2. **"Un câmp viu, conversațional, amestecat, cu facts incomplete, corecții, schimbări de topic și receipts diverse produce advisory și residue, dar nu produce automat ReferenceField."** — Confirmed across 3 attempts.

The root cause is not in BYON code — it is in the **mismatch between FCE-Ω's expected input regime and what a real conversational pipeline produces**:

- FCE-Ω's coagulation criterion assumes input where coherent repetition does NOT accumulate residue. In its theoretical regime, identical inputs are "no new information" → Z stays bounded.
- BYON's symbolic slot_event writes appear to be counted as fresh events by the observer, accumulating Z regardless of semantic similarity.

Without modifying FCE-Ω internals (vendor-locked source, BSD-3-Clause but architecturally separate), the assimilation logic cannot recognize "coherent repeat = no residue".

### What CANNOT be done without modifying FCE-Ω

- Make Z decay between events.
- Distinguish contradiction residue from repetition residue inside the observer.
- Change the AR/κ/I_t/B_t formula.

### What we COULD still try (open questions)

1. **Rolling center summary at BYON level.** Periodically replace N coherent events with ONE distilled "anchor fact" stored back as a single FCE write. This reduces effective event rate without losing information. Estimated effort: ~1 day.

2. **Multi-perspective observer fan-out.** `fce_multiperspectival_enabled=True` is set but unexplored. The multi-perspective layer may decouple Z accumulation per perspective. Estimated effort: dive into vendor source.

3. **Symbolic-to-numerical-ratio tuning.** v0.6.3 added numerical companion writes; tf_engine bank is fed but observer may not see it. A different bridge configuration (`fce_translator.py`) might route numerical entries into the observer's perspective field. Estimated effort: requires understanding FCE-Ω observer internals.

4. **Acknowledge architectural compatibility.** The FCE-Ω theory may need a separate "natural language conversation regime" — not just default coagulation parameters but a different coagulation pathway. This is a **research contribution to FCE-M itself**, not a BYON fix.
